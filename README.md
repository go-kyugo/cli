# kyugo CLI
Usage

```bash
./kyugo create <type> <name>
```

Tipos suportados: `controller`, `model`, `repository`, `service`, `middleware`, `migration`, `seed`, `dto`, `validation`.

Exemplo:

```bash
./kyugo create model user
```

Isso cria o arquivo `models/user.go` com um stub de struct.
Tipos suportados: `controller`, `model`, `repository`, `service`, `middleware`, `migration`, `seed`, `dto`, `validation`.

Exemplo:

```bash
./kyugo generate model user
```
